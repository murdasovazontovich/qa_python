# qa_python
В проекте реализованы автотесты для класса BooksCollector, проверяющие корректность работы с книгами, жанрами и избранным.

1. Добавление книг
Проверяется возможность добавления новых книг с корректным названием.
Проверяется, что несколько разных книг успешно добавляются.
Проверяется, что книга не добавляется:
с пустым названием;
с названием длиной более 40 символов.

2. Работа с жанрами
Проверяется, что для существующей книги можно установить жанр.
Проверяется, что по умолчанию у новой книги жанр отсутствует.
Проверяется, что запрос жанра несуществующей книги возвращает None.
Проверяется, что метод get_books_genre() возвращает корректный словарь книг и жанров.
Проверяется, что метод get_books_with_specific_genre() возвращает только книги заданного жанра.

3. Книги для детей
Проверяется, что книги с детскими жанрами (например, Фантастика) попадают в список книг для детей.
Проверяется, что книги с возрастными жанрами (например, Ужасы) не попадают в список книг для детей.

4. Избранные книги
Проверяется возможность добавления книги в избранное.
Проверяется корректное удаление книги из избранного.
